# Sendy Messenger (Android)

**Sendy** — это приватный Android-мессенджер с упором на безопасность, стабильность и контролируемый доступ.  
Репозиторий предназначен исключительно для публикации информации и релизов приложения.

---

## 🔒 О проекте

Sendy — это закрытый мессенджер, доступ к которому предоставляется только ограниченному кругу пользователей.  
Приложение разработано для безопасного общения с базовым, но надежным функционалом.

---

## ✨ Основные возможности

На текущий момент Sendy поддерживает:

- 💬 Личные чаты  
- 👥 Групповые чаты  
- 📝 Отправка текстовых сообщений  
- 🖼️ Отправка изображений  
- 🎥 Отправка видео  
- 🎤 Отправка голосовых сообщений (кружков)  
- 📁 Передача файлов  
- 👍 Реакции на сообщения  

Функционал постепенно развивается и расширяется.

## 🛡️ Безопасность

Безопасность является приоритетом в Sendy. Реализованы следующие меры защиты:

### 🔐 Сквозное шифрование трафика
Весь пользовательский трафик зашифрован на протяжении всего пути до конечного backend-сервера.  
Это исключает возможность перехвата и чтения данных третьими лицами.

### 🌐 Использование прокси-серверов
Для дополнительной защиты инфраструктуры используется прокси-сервер:

- Скрывает реальные backend-узлы  
- Помогает предотвращать сетевые атаки  
- Повышает устойчивость к DDoS и другим угрозам  

### 🔁 Регулярная смена токенов
Система безопасности предусматривает регулярную ротацию токенов аккаунта:

- Снижает риск компрометации сессий  
- Повышает общий уровень защиты пользователей  
- Обеспечивает актуальность авторизации
